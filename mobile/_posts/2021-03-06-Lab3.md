## Lab 3

### Challenge 1
* Use Lab01.apk which can be found on Leho
* Use __APKtool__ to find a flag in the code

**Solution:**
* Download APKtool
* Optional: set the environment path for APKtool
* run APKtool: `apktool d lab01.apk` (d = decompile)
* go to the directory lab01
* look into the res/values folder
* open the strings.xml file
* look for all "odd" strings
* you can find `flaglab01` xml key and its value there






### Challenge 2
* Use Lab02.apk which can be found on Leho

**Solution:**
* Download jadx-gui
* open jadx-gui and open lab02.apk
